# Fonts

Drop these three variable/regular `.woff2` files here for the theme to use its custom typography. The theme falls back to system fonts if they're missing, so nothing breaks — you just lose the branded look.

| File                             | Source                                             |
|----------------------------------|----------------------------------------------------|
| `anton-regular.woff2`            | https://fonts.google.com/specimen/Anton            |
| `space-grotesk-variable.woff2`   | https://fonts.google.com/specimen/Space+Grotesk    |
| `inter-variable.woff2`           | https://rsms.me/inter/                             |

### Getting the `.woff2` files

Easiest: use **google-webfonts-helper** — https://gwfh.mranftl.com — select the family, choose "Modern Browsers", download the ZIP, rename the file to match the table above.

For Inter, download the variable `Inter.var.woff2` from rsms.me/inter/ and rename to `inter-variable.woff2`.
